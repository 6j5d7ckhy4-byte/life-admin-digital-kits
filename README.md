# Life Admin Digital Kits

A low-risk digital product and micro-publishing project focused on practical checklists, templates, and starter kits that help people organize everyday life admin.

## First product line

**Life Admin Starter Kit** — simple, beginner-friendly worksheets for bills, documents, taxes, appointments, subscriptions, household tasks, and family responsibilities.

## Important disclaimer

These materials are for personal organization and education only. They are not legal, tax, medical, or financial advice. Users should consult qualified professionals for professional guidance.

## Current assets

- `index.html` — public landing page
- `products/30-minute-life-admin-reset.md` — first free asset draft
- `operating-plan.md` — 30-day project plan and weekly operator workflow
